# Simple Calculator

A functional desktop calculator application built with Python and Tkinter. This project focuses on event-driven programming, handling dynamic string evaluations, and basic UI grid layout management.

## 🚀 Key Features

- **Standard Operations:** Supports addition, subtraction, multiplication, and division.
- **Smart Arithmetic Routing:** Automatically evaluates pending expressions if a new operation key is pressed sequentially.
- **Robust Error Handling:** Integrated exception safety loops to handle `ZeroDivisionError` and invalid data syntax smoothly with native alert modals.
- **Clean Display Rules:** Prevents multiple zero inputs and auto-clears structural string overflows.

## 🛠 Tech Stack

- **Language:** Python 3.11+
- **GUI Framework:** Tkinter
- **Architecture:** Functional layout splitting with dynamic `lambda` callback triggers.
